# 🩺 MedTrack

A modern Flutter prototype designed to help doctors and small clinics manage **patient records** and **appointments** in a simple, organized, and responsive way.

---

## ✨ Overview

MedTrack is a lightweight clinic assistant prototype built with Flutter.  
It helps doctors digitize basic daily tasks such as:

- registering new patients
- storing medical notes and conditions
- scheduling appointments
- reviewing patient and appointment lists

Instead of relying on paper records, MedTrack offers a cleaner and faster workflow for small clinics.

---

## 🚀 Features

- 👤 Add new patients
- 📋 View patient records
- 🩺 Store medical conditions and notes
- 📅 Schedule appointments
- 🕒 View appointment list
- 💻 Responsive UI for mobile and web
- 🎨 Clean and modern interface

---

## 🎯 Project Goal

This project was built as a **technical challenge prototype** to demonstrate:

- clean Flutter UI design
- structured and readable code
- responsive layout
- practical problem-solving for real clinic workflows
